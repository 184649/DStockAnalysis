# バフェット採点 算出方法（独自アルゴリズム）

バフェットの思考(長期で稼ぎ続けられるか・競争優位・財務安全・利益とCFの安定・資本効率・株主還元の合理性・価格妥当性)
を総合して **100点満点**で評価する。単なる低PER/低PBRランキングではなく、**割安性より「事業の質」を重視**する。

- 実装: `DStockAnalysis.Core/Services/BuffettScoreCalculator.cs`(`Calculate(Stock) → BuffettResult`)
- 結果: `DStockAnalysis.Core/Models/BuffettResult.cs`(総合点・6サブスコア・データ信頼度・評価ランク・判定)
- **外部アクセスは一切なし。** Stock に既にある指標(CSV取込・自動取得済み)だけで採点する。

## 総合スコアの重み

```
BuffettScore = 事業耐久力 ×0.25 + 収益力 ×0.20 + 財務安全性 ×0.15
             + 成長安定性 ×0.15 + 資本配分 ×0.10 + 割安性 ×0.15
```

| サブスコア | 重み | 主な指標 |
|---|---|---|
| 事業耐久力 BusinessDurability | 25% | 営業利益率・長期売上成長・営業利益の安定・営業/フリーCF・堀の代理(高利益率/高ROE/長期非減配) |
| 収益力 Profitability | 20% | ROE(借入嵩上げは割引)・ROA・営業利益率・純利益率・営業CFマージン |
| 財務安全性 Safety | 15% | 自己資本比率・有利子負債比率・FCF・配当余力・営業CF(**金融業は専用基準**) |
| 成長安定性 GrowthStability | 15% | 売上/営業利益/EPS成長・長期成長・下方耐性(減配の少なさ) |
| 資本配分 CapitalAllocation | 10% | 連続増配・増配率・配当性向の健全さ・自社株買い・総利回り |
| 割安性 Valuation | 15% | PER・PBR・MIX(PER×PBR)・FCF利回り・総利回り(高品質は補正) |

各サブスコアは指定された閾値で 0〜100 点に換算し、重み付き平均する(仕様の各閾値表に準拠)。

## 欠損値の扱い(重要)

- **未取得指標を0点扱いしない。** 欠損は評価対象から除外し、残った指標で**重みを再配分**する。
  (本モデルは数値型のため、値が 0 のものを欠損とみなす。成長率・CF等の負値は実値として採点。
   ただし有利子負債比率0=無借金は満点、配当継続性0は下限点として扱う。)
- **データ信頼度 DataConfidence** = 取得済みの重要指標数 / 重要指標数 × 100
  (重要指標: PER, PBR, ROE, EquityRatio, RevenueGrowth5Y, OperatingProfitGrowthRate, OperatingMargin,
   DividendYield, PayoutRatio, OperatingCF, FreeCashFlow, MarketCap, EPS, BPS の14個)
- 信頼度による上限: ≥80→補正なし / ≥60→85 / ≥40→75 / <40→60。

## 強制上限(複数該当時は最小を採用)

| 条件 | 上限 |
|---|---|
| IsSampleIndicators=true(サンプル指標) | 40 |
| PER≤0 または EPS≤0(赤字) | 65 |
| 非金融業で 営業CF<0 かつ フリーCF<0 | 60 |
| 非金融業で 自己資本比率<20% | 60 |
| 配当性向>100% かつ 配当利回り≥4% | 70 |
| 売上5年成長<0 かつ 営業利益成長<0 | 65 |
| ROE<5% かつ 営業利益率<5% | 60 |

## 評価ランクと判定

| スコア | ランク |
|---|---|
| 90–100 | S | 80–89 | A | 70–79 | B | 60–69 | C | 50–59 | D | 0–49 | E |

`JudgementText` はランクに応じた日本語コメント + 最も低いサブスコアを弱点として補足する。

## 重みの教師データ最適化(説明可能な学習)

6サブスコアを総合点へ合成する重みは、固定の感覚値ではなく**教師データへ適合するよう最適化**する
(ニューラルネットの学習に近いが、外部MLライブラリは使わず説明可能)。

- **教師データ**: `BuffettScoreCalibrationSet`(Claude作成・45件・10カテゴリ: 超優良S/優良A/総合商社/ヒット依存/
  安定低成長/低PER低収益/赤字危険/高配当危険/金融/景気敏感)。各件に期待スコア範囲・許容ランク・S禁止・危険フラグ・根拠を付与。
- **Loss**: スコア誤差²(期待範囲外のみ)＋ランク不一致(200)＋S禁止違反(1000)＋危険銘柄が70点以上(1000)。
- **最適化**: `BuffettScoreWeightOptimizer` が制約(合計1.0/各範囲/耐久力+安全≥0.35/割安≤0.20)を満たす重みを
  グリッド探索し、TotalLoss 最小の重みを `DefaultWeights` として採用(初回のみ計算しキャッシュ)。
- **採用重み(実測)**: 事業耐久力0.20 / 収益力0.12 / 財務安全0.25 / 成長安定0.15 / 資本配分0.15 / 割安0.13。TotalLoss≈39。

ランク S は点数(90+)だけで決めず、各サブスコア・財務・CF・**長期成長(10年)≥3%**・データ信頼度≥85 等の厳格条件を
すべて満たす場合のみ許可(商社・市況株・ヒット依存はS不可)。個別分析に**高評価要因・減点要因・ランク判定理由**を表示する。

## 業種プロファイル(StandardCompany / FinancialCompany / TradingCompany)

業種特性で採点プロファイルを切り替える(特定銘柄コードのハードコードはしない)。

- **TradingCompany(総合商社・卸売業)**: Sector に「卸売」を含む、または Name/Theme/Description に「総合商社・商社・卸売」を含む。
  商社は営業利益率が低く見え、売上が市況・会計で大きくぶれるため、**営業利益率の絶対値評価を弱め**、
  ROE・キャッシュフロー・財務安全性・株主還元・PER/PBR/配当性向を重視して採点する。
  - 事業耐久力 = ROE品質×0.25 + 利益とCFの安定×0.25 + 財務耐性(自己資本)×0.20 + 株主還元の安定×0.15 + 価格妥当性×0.15
  - 収益力 = ROE×0.45 + 純利益率×0.15 + 経常利益率×0.15 + 営業CF×0.15 + ROA×0.10(営業利益率の絶対値は使わない)
  - 成長安定性 = EPS成長×0.25 + 純利益成長×0.25 + 経常利益成長×0.20 + 増配率×0.15 + 下方耐性×0.15(売上成長で強く減点しない)
  - 割安性は PER≤15・PBR≤2.0・ROE≥12・性向≤50 で +8、資本配分は健全配当(性向25-45%等)で下限70。
  - 例(伊藤忠相当: ROE14.6%・自己資本39%・PER13.5・性向32%): 総合 **B(約72)**(従来Dから是正)。

## 優良・妥当価格の下限補正

データ信頼度≥80・PER∈(0,18]・PBR∈(0,2.5]・ROE≥12・自己資本≥30・配当性向∈[20,60]・非サンプル を満たし、
かつ失格条件(EPS≤0/PER≤0/ROE≤0/自己資本<20/営業CF&FCF両赤字/性向>100/サンプル/信頼度<80)に該当しない場合、
**BuffettScore の最低値を 70(=B以上)**に補正する(無条件の高評価ではなく、明確な弱点が無い優良株の沈み込みを防ぐ)。

## 金融業の扱い

業種(Sector)に「銀行・保険・証券・その他金融・金融」を含む場合は金融業として扱い、**財務安全性は専用基準**
(自己資本比率・ROE安定・経常/純利益成長・配当性向・減配耐性)で評価する。営業CF・有利子負債比率・FCF は使わない。

## UI / API

- 一覧: バフェット総合点・評価ランク・データ信頼度・6サブスコアを列表示。
- 個別分析: 「バフェット採点」ボックス(総合点・ランク・信頼度・判定 + 6サブスコアのバー)。
- 比較: 総合点・6サブスコア・信頼度・ランクを横並び比較。
- API: `/api/screen`・`/api/compare` の `StockSummaryDto.Buffett`、`/api/stocks/{code}` の `stock.Buffett` に
  `BuffettResult`(総合点・6サブスコア・DataConfidence・OverallGrade・JudgementText)を含む。
- CSV取込・自動取得で指標が更新されると、スコア再計算時にバフェット採点も更新される。
